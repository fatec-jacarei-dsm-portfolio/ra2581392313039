Bicicreta

O projeto tem como objetivo desenvolver uma ferramenta para facilitar o aluguel de bicicletas. Após sua conclusão, os usuários podem disponibilizar suas bicicletas para aluguel por meio de anúncios, buscar bicicletas disponíveis no sistema para locação e realizar avaliações, que contribui para a criação de um sistema de reputação tanto para locadores quanto para locatários.

Vintura

O Vintura é um software de gerenciamento de projetos voltado para Geoprocessamento, com o objetivo de centralizar os colaboradores da empresa Visiona em um único ambiente. Dessa forma, eles poderão realizar suas tarefas individuais, acompanhar seu progresso e receber feedbacks, agilizando as entregas da empresa. Os projetos serão apresentados a cada colaborador conforme seu papel específico, evitando qualquer interferência entre eles.

Acesse o projeto

Da Vinci Care

O aplicativo foi desenvolvido para atender às necessidades dos usuários, oferecendo duas opções de dieta: uma personalizada, com o auxílio de um profissional de saúde, e outra com dietas predefinidas para ganho de massa, perda de peso e manutenção. As dietas serão adaptadas para diferentes faixas etárias. Além disso, o aplicativo contará com funcionalidades como um checklist diário para registrar os alimentos consumidos, calcular automaticamente as calorias ingeridas e permitir a adição de novos alimentos. Isso proporcionará uma experiência mais completa e personalizada, facilitando o acompanhamento das escolhas alimentares.
